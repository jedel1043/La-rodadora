Proyecto de chatbot para mueso La Rodadora.

# Requisitos (Python)
- Python 3+
- kivymd
- kivy
- tensorflow
- keras
- numpy
- nltk
- TTS
- speechrecognition
- pyaudio

# Requisitos del sistema
- Poetry
- CUDA (si se requiere aceleración de GPU)

# Ejecución

```bash
poetry install
poetry run python src/gui/main.py
```
